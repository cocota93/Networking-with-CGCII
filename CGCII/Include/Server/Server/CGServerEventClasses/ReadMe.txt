1. CGCII Server Event Classes

  서버 운영 시 특정한 시간 혹은 일정시간 마다 수행해야하는 Event가 있을 경우 이를
 위한 클래스들을 의미한다.
  일반 적으로 서비스를 운영할 때 특정 시간에 서버를 중지시키거나 백업을 하거나 
 접속자를 조사해서 접속해제시키거나 등등 특정한 시간 혹은 일정 시간마다 실행되어야
 하는 운용상 필요한 실행을 위해 사용된다.

   Event Classes들은 크게 한번만 실행되는 객체, 반복적으로 실행되는 객체, 시작과
  종료 시점에 각각 실행되는 객체, 임의의 정해진 시간마다 실행되어야 하는 객체 등이
  존재한다.
   필요에 따라 해당 객체를 생속받앙 재정의 하거나 생성하여 Event Manager에 등록
  함으로써 실행되게 된다.
  
2. CGEvnet::Manager와 ICGEventObject 클래스
    CGEvent::Manager는 EventObject를 실행시키는 통합관리자로서 생성된 Event Object는
   반드시 Event Manger에 등록되어야 실행된다.
    Event Manager는 싱글톤 객체로 따로 생성할 필요 없이 내부적으로 유일하게 하나만
   생성되어 있다.

3. CGEvent::Object 클래스들

   1) 단한번 실행
      - CGEvent::Object::COnce
      - CGEvent::Object::COnceFunction<TFUNCTION>
   2) 반복적으로 실행
      - CGEvent::Object::CIteration
      - CGEvent::Object::CIterationFunction<TFUNCTION>

   3) 시작과 끝나는 시점에 실행
      - CGEvent::Object::CStartEnd
      - CGEvent::Object::CStartEndFunction<TFUNCTION>

   4) 시작시점과 끝시점에 실행되며 그 사이 시간에는 일정시간마다 실행
      - CGEvent::Object::CStartIterationEnd
      - CGEvent::Object::CStartIterationEndFunction<TFUNCTION>

4. CGEvent Schedule 관련 클래스와 기능
   Schedule은 여러 개의 실행을 하나의 객체에서 실행할 수 있도록 설계되었다.
   또 앞의 Event 객체와 달리 따로 CSchedule 객체를 생성하여 이를 Event 객체에
   등록하여 사용하도록 되어 있다.

   1) CGEvent::Object::CSchedule 클래스

   2) CGEvent::Object::CScheduleFunction 클래스

   3) CGEvent::Schedule 클래스
      CGEvent Schedule객체에 등록되어 사용되는 객체이다.
	  생성후 필요한 값을 설정하고 이를 Event Schedule 객체에 등록한다.
      - CGEvent::Schedule::CFunction
      - CGEvent::Schedule::CLambda
      - CGEvent::Schedule::CMemberFunction<TCLASS>
